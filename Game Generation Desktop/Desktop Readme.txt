How to create a lesson:
Click on "Create New Lesson".
In order to add new slides click on "Add New Slide", and choose the type of slide you want to create.
If you want a picture slide, you can add pictures by clicking "Choose Picture" and choosing a picture from a local directory.
If you want a game slide, choose "Game Slide" as the type of the new slide, and choose the game that you want from the drop down meny that opens.
In order to save the game, click on "Save Lesson".
The lesson will be saved in xml format in xmlDir directory in the same directory of the .jar file.
In order to view the lesson on your tablet, copy manually the xml file that was created onto your tablet.
